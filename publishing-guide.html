⚡ Quick Reference – Per-Post Workflow (5 Steps)

Complete each post in ~10 minutes by following this exact order.
1
▶ Upload to YouTube
~3 min per video
▶
  • Go to studio.youtube.com → Create → Upload video
  • Select your recorded video file
  • Title format: [Post Title] | KJV Right Division | [Verse]
  • Description: paste the "About this study" block (see template below)
  • Tags: KJV, right division, dispensational, Bible study, 2 Timothy 2:15, [topic keywords]
  • Thumbnail: upload a custom gold/dark text thumbnail OR use auto-generated
  • Visibility: Public (or Unlisted until blog post is ready)
  • Copy the video URL from the browser bar once upload finishes
https://www.youtube.com/watch?v=XXXXXXXXXXX
💡 For YouTube Shorts (under 60 sec), the URL format is youtube.com/shorts/XXXXXXXXXXX. The embed mapper handles both formats automatically.
2
🔗 Map Video to Post (YouTube Embed Mapper)
~1 min
▶
  • Open youtube-embed-map.html
  • Click the matching post number in the left panel (e.g., Post #141)
  • Paste the YouTube URL you just copied into the URL field
  • Click "Generate Embed Code"
  • Click "Copy" on the "Full blog post section" code block
The "Full blog post section" code is ready to paste directly into Blogger's HTML editor — it includes the iframe with proper attributes and a subscribe link.
3
📝 Publish on Blogger
~4 min
▶
  • Go to blogger.com → Rightly Dividing KJV → New post
  • Title: copy the post title from the HTML file (e.g., "Post #141: The Rapture vs Second Coming Distinctions")
  • Click the HTML editor tab (⟨⟩ icon) — NOT the compose tab
  • Open the matching week HTML file (e.g., week6-posts.html) in your browser
  • Find Post #NNN → Select All HTML from the <article> tag to the closing </article>
  • Paste into Blogger's HTML editor
  • Find [VIDEO EMBED – Replace with YouTube iframe: YOUR_YOUTUBE_LINK_NNN]
  • Replace that placeholder with the embed code you copied from the YouTube Mapper
  • Add Labels: the week name (e.g., "Week 6"), topic category, "KJV", "Right Division"
  • Set permalink to match the post title slug
  • Preview → Publish
⚠ Always use the HTML editor in Blogger — the compose/visual editor may strip custom CSS classes and div attributes from the post HTML.
4
📊 Mark Complete in Dashboard
~30 sec
▶
  • Open content-dashboard.html
  • Find the post by number (use search or week filter)
  • Click the YouTube toggle (○ → ✓) for the YouTube column
  • Click the Blog toggle for the Blogger column
  • Paste the YouTube URL into the URL field on that row
  • Stats bar at top auto-updates percentage complete
📌 The dashboard state resets on page reload (in-memory only). For permanent tracking, copy the completed post list to a text file or spreadsheet at the end of each session.
5
🌐 (Weekly) Update GitHub Pages
~5 min per week
▶
  • When a full week's posts are published on Blogger, copy the Blogger post URLs
  • Open index.html (the GitHub landing page) in a text editor
  • Find the week's blog link placeholders and update them with the real Blogger URLs
  • Commit and push to the rightlydividingkjv GitHub repo
  • GitHub Pages auto-deploys within 1–2 minutes
  • Verify at henrynkoh.github.io/rightlydividingkjv/
git add index.html && git commit -m "Week N: update Blogger post links" && git push

🔍 SEO & Growth Tips

YouTube: The first 125 characters of the description appear in search results. Start with the verse text, not with "In this study…"
Blogger: Use the post title as the permalink slug exactly (Blogger does this by default). Keep the slug to 5–8 words max.
Cross-linking: Every YouTube description links to the blog post. Every blog post embeds the YouTube video. This cross-traffic compounds over time.
Consistency: Publishing 4 posts/day to YouTube Shorts (under 60 sec each) gives 28 posts/week — the fastest path to completing all 232 posts while keeping the series cohesive.
Verse-first strategy: Title videos with the verse first ("1 Thessalonians 4:16-17 – The Rapture") for better searchability when people look up a specific Bible passage on YouTube.
